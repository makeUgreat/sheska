# 문서 컨벤션 인덱스

## 참조 기준

영어 문서가 기준이다.
한글 문서는 사람이 참고하기 위한 번역 문서다.

## 읽기 규칙

현재 작업과 관련된 문서만 읽는다.
기본적으로 모든 문서를 읽지 않는다.

## 라우팅

- 문서를 생성, 수정, 리뷰할 때: [문서 작성 가이드라인](./documentation-guidelines.md)을 읽는다.
- 이 문서 컨벤션 인덱스를 수정할 때: [문서 컨벤션 인덱스 라우터](./documentation-index-router.md)를 읽는다.

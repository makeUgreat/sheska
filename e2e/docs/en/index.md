# E2E Convention Index

## Synchronization Policy

English and Korean `e2e` convention documents are paired documents that should describe the same policy.
When they conflict, choose the intended policy from either language and update both documents in the same change unit.

## Reading Rules

Read only the `e2e` convention documents relevant to the current task.
When changing public project Markdown documents, also read the repository documentation convention index.

## Routing

- Writing, reviewing, or deciding scope for E2E test files: read [E2E Test Convention](./test.md).
